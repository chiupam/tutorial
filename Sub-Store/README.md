## 高级订阅管理工具

Loon 和 Surge 的订阅管理神器，可以在本地对机场订阅进行各式自定义的修改

### 功能

- 已经开放或部分开放

  - 格式转换

  - 订阅过滤（部分开放）

- 仍未开放

  - 重命名订阅节点

  - 排序订阅

### 优点

- 高度自定义订阅节点

- 本地解析，完全不会泄露订阅链接

  - [在 Loon 中订阅节点](https://github.com/chiupam/tutorial/blob/master/Sub-Store/Configuration.md)
